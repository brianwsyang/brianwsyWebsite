AirBnB Price Prediction

Fall 2019: Data Analytics & Mining Class Group Project

Goal: Using the dataset available at insideairbnb.com, predict the price of a new AirBnB listing so that the host can refer to this value when they're offering their place on the site.
At the time of this project, we used the dataset that encompassed all the months from January 2018 to August 2019 to predict the price for September 2019.

GitHub: github.com/brianwsyang/airbnbPricePrediction

Slide Deck: FIND LINK

Python: pandas, numpy, scikit-learn

Next Steps: Possible next steps for this project would be to use a bigger dataset and remove less features. Bigger dataset means more data to train on to improve prediction accuracy. And while pre-processing the data, we had to remove a lot of features because there were a lot of redundant or empty values that weren't as statistically relevant to the outcome.
