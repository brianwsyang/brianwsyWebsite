Maroon 5 Photos

Fall 2019: Class Personal Project

Goal: For my photography class, the assignment was to choose a song and take photos to accompany the lyrics. And I decided to do my assignment on Maroon 5's Sunday Morning.

While doing this assignment, I thought it would be interesting to showcase my photos on a website since I was just starting to learn web design and development.

GitHub: github.com/maroonFivePhotos

Preview: preview of the website

HTML

CSS: Flexbox

Updates: Added a play button that will play the song in the background. The button is interactive so it can pause and play the song.

Next Steps: Something I would like to do in the future with this project would be to recreate the website with ReactJS and add some interactive elements. When I was working on this project, I had only learned HTML and CSS, so I didn't quite know how to build a website with ReactJS. One of the interactive elements that I thought about adding was a carousel that would rotate through based on the song that will be playing in the background.
